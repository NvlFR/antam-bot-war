# Security Policy

## Supported Versions

We actively maintain the following versions:

| Version | Supported          |
| ------- | ----------------- |
| main    | :white_check_mark: |  <!-- main branch selalu up-to-date -->
| < main  | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please responsibly report it by emailing:

**Email:** security@nvlfr.dev  <!-- ganti sesuai kontak kamu -->

Please include the following details to help us reproduce and fix the issue:
- Steps to reproduce
- Screenshots or logs (if applicable)
- Your suggested fix (optional)

We aim to respond to all security reports within 48 hours. We may reach out for clarification or additional testing details.

## Response Process

1. Confirm receipt of your report.
2. Assess the severity and impact of the vulnerability.
3. Work on a fix and release a patched version.
4. Notify the reporter when the fix is released.
